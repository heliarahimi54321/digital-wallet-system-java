# Digital Wallet System

A console-based digital wallet system developed in Java. This project allows users to manage wallet balance, add money, spend money, and handle transaction fees through a simple command-line interface.

## Features

* Add money to wallet
* Spend money from wallet
* Show current balance
* Apply transaction fee on spending

## Technologies Used

* Java
* Object-Oriented Programming
* Console-based interface

## Purpose

This project was built to practice Java fundamentals, class design, and transaction logic.
